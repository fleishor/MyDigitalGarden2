# Creating a new pane

## Horizontal pane

<kbd>alt</kbd> + <kbd>shift</kbd> + <kbd>+</kbd>

## Vertical pane

<kbd>alt</kbd> + <kbd>shift</kbd> + <kbd>-</kbd>

# Switching between panes

<kbd>alt</kbd> + <kbd>arrow keys</kbd>

# Resizing a pane

<kbd>alt</kbd> + <kbd>shift</kbd> + <kbd>arrow keys</kbd>

# Closing a pane

<kbd>alt</kbd> + <kbd>shift</kbd> + <kbd>w</kbd>
